WSU WDS Sidecar

WordPress cleanup utilities.

Page/Post

Update No DataOpen visible Posts or Pages marked No Data in the Accessibility column and save them so WordPress can regenerate checks.

Open Visible ListOpen every visible item on the current WordPress list screen in a background tab.

Headings

Fix Heading OrderChange content heading blocks to H2 to clear incorrect heading order alerts. The post/page title is not touched.

Make All H2Convert all content heading blocks to H2. The post/page title is not touched.

Unbold HeadingsRemove bold markup from heading block text while preserving heading level and classes.

Change Heading LevelChange all heading blocks of one selected level to another level while preserving content and classes.

Select Heading BlocksList individual heading blocks so selected items can be changed to a chosen heading level.

Set H2 SizeApply a WSU font-size class to every H2 block while preserving unrelated Advanced classes.

Alt Tags

Set Linked Image AltFor images linked directly to image files, set alt text that describes the full-size file destination.

Suggest Alt TextScan image blocks with missing or suspicious alt text, generate editable suggestions, and apply only checked items.

Links

URL Link TextFind links whose visible text is a URL and replace that text with the linked page title.

Generic Link TextFind generic link text like here, click here, and read more, then replace it with the linked page title.

Tracked LinksReplace URLDefense and Outlook Safe Links wrapper URLs with the decoded original destination URL.

New-Tab LinksRemove open-in-new-tab behavior from rich text links while preserving other link settings.

Text Cleanup

Bold Paragraph FixesFix paragraphs that are entirely bold, including converting short bold lines into H2 headings when appropriate.
