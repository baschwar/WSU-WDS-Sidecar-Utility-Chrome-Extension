WSU WDS Sidecar

WordPress cleanup utilities.

Page/Post

Update No Data ItemsOpen visible Posts or Pages marked No Data in the Accessibility column and save them so WordPress can regenerate checks.

Headings

Fix Heading OrderChange content heading blocks to H2 to clear incorrect heading order alerts. The post/page title is not touched.

Make All Headings H2Convert all content heading blocks to H2. The post/page title is not touched.

Unbold HeadingsRemove bold markup from heading block text while preserving heading level and classes.

Set H2 SizeApply a WSU font-size class to every H2 block while preserving unrelated Advanced classes.

Alt Tags

Set Linked Image AltFor images linked directly to image files, set alt text that describes the full-size file destination.

Suggest Alt TextScan image blocks with missing or suspicious alt text, generate editable suggestions, and apply only checked items.

Links

Replace URL Link TextFind links whose visible text is a URL and replace that text with the linked page title.

Replace Generic Link TextFind generic link text like here, click here, and read more, then replace it with the linked page title.

Unwrap URLDefense LinksReplace urldefense.com wrapper URLs with the decoded original destination URL.

Remove Open-In-New-TabRemove open-in-new-tab behavior from rich text links while preserving other link settings.

Text Cleanup

Bold Paragraph FixesFix paragraphs that are entirely bold, including converting short bold lines into H2 headings when appropriate.
