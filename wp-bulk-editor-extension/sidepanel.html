WSU WDS Sidecar

Bulk utilities for the WordPress editor.

Update No Data Posts/Pages

No Data UpdaterOpen visible Posts or Pages marked No Data in the Accessibility column and save them so WordPress can regenerate checks.

Accessibility Fixes

Heading OrderChange content heading blocks to H2 to clear incorrect heading order alerts. The post/page title is not touched.

URL Link TextFind links whose visible text is a URL and replace that text with the linked page title.

New Tab LinksRemove open-in-new-tab behavior from rich text links while preserving other link settings.

Linked Image Alt TextFor images linked directly to image files, set alt text that describes the full-size file destination.

Alt SuggestionsScan image blocks with missing or suspicious alt text, generate editable suggestions, and apply only checked items.

Bold ParagraphsFix paragraphs that are entirely bold, including converting short bold lines into H2 headings when appropriate.

Heading CleanupConvert all content heading blocks to H2. The post/page title is not touched.

Heading SizeApply a WSU font-size class to every H2 block while preserving unrelated Advanced classes.
